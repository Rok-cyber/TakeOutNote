리스트 결과를 AJAX를 이용해 받아와서 listDesign.js에서 정의된 아래 함수를 호출후 내용이 나타남

rtnCell(obj,uid,image,link,dragd,name,icon,loc,price,rese,ccnt,tag,sout,cp_price,rank,cooperate,cate) : 리스트형일 경우
rtnImg(uid,image,link,dragd,name,icon,price,ccnt,sout,cp_price,rank,cooperate,cate) : 이미지형일 경우

obj : list_after (id)
uid : 등록 uid
image : 상품 이미지
link : 상품 상세 페이지 링크주소
dragd : 드래그정보
name : 상품명
icon : 상품 아이콘
loc : 상품위치(분류)
price : 상품가격
rese : 적립금
ccnt : 이용후기 등록 수
tag : 등록 태그
sout : 품절시 값 : 1

cp_price : 쿠폰적용가

rank : 베스트 상품 페이지에서만 사용 순위

cooperate : 공동구매 상품일때 값 : 1

cate : 분류번호

해당 정보를 받아서 DOM형태로 코드 생성 후 출력
