리스트 결과를 AJAX를 이용해 받아와서 아래 함수를 호출후 내용이 나타남

rtnRowAfter(obj,num,point,title,name,buy,date,content,mod,uid,name2,link,image,dragd)
obj : list_after (id)
num : 출력 번호
point : 평점
title : 제목
name : 이름
buy : 구매자가 등록했을 경우
date : 등록일
content : 내용
mod :수정일 경우
uid : 등록 uid
name2 : 상품명
link : 상품 상세 페이지 링크주소
dragd : 드래그정보

rtnRowQna(obj,num,title,name,ans,date,content,answer,mod,uid)
obj : list_after (id)
num : 출력 번호
title : 제목
name : 이름
ans : 답변유무
date : 등록일
content : 내용
answer : 답변내용
mod :수정일 경우
uid : 등록 uid


해당 정보를 받아서 DOM형태로 코드 생성 후 출력
