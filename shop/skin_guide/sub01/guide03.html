스타일정의(style.css)
스타일 정의중 아래 정의 구역은 반드시 있어야 됩니다.
  • common default  // 기본 스타일 정의
  • paging  // 페이징에 사용되는 스타일
  • Ranking  // 검색어 순위에 사용되는 스타일
  • money Bar  // 가격검색 바에 사용되는 스타일
  • Button Effect  // 텍스트 버튼에 사용되는 스타일
  • LightBox Css  // LightBox에 사용되는 스타일
  • MessageBox CSS  // Message Box에 사용되는 스타일
  • TAG CLOD  // Tag Clod에 사용되는 스타일
