스킨 파일 구성
기본 스킨 파일은 skin/default 폴더에 있으며 스킨 파일의 구성은 총 73개의 파일로 아래와 같습니다.
이미지는 img폴더를 생성 후 사용하시면 되고 img 폴더에 skin_thum.gif 썸네일 파일을 올려 놓으면 관리자 페이지 스킨 선택에 나타 납니다.
스킨 파일을 제작 후 skin 폴더 안에 xxx폴더를 생성 후 파일을 올리시면 관리자 페이지에서 해당 폴더명을 선택 하면 해당 스킨이 적용이 됩니다.
