분류코드 매칭 설정
  * 타서버 이미지 복사   상품 상세내역에 링크된 타서버 이미지를 본 쇼핑몰에 저장 후 링크를 변경하는 기능입니다.
! 엑셀(CSV)파일 형식
구분코드,분류코드, 상품명, 소비자가, 판매가, 제조사, 원산지, 브랜드, 모델명, 적립금, 재고, 큰이미지, 중간이미지, 리스트이미지, 작은이미지,
디스플레이 이미지, 이미지자동생성, 상품 다른 이미지, 상품상세설명, 상품옵션
! 엑셀로 작업하셔서 다른이름으로 저장을 누르시고 파일형식을 CSV(쉼표로분리)로 선택하시고 저장하시면 됩니다.
! 대량상품 등록일 경우 서버 부하 및 처리 시간이 많이 걸릴 수 있으니 분할해서 처리 하시기 바랍니다.
! 분류코드매칭을 사용하여 등록 할 경우 먼저 '분류코드 매칭 설정'에서 설정된 매칭정보를 이용하여 분류코드가 자동으로 등록 됩니다.
! 엑셀샘플파일을 다운 받으시고 참고 하시기 바랍니다. [엑셀샘플파일 다운받기]
일괄처리
상품정보 엑셀 작성 안내
  * 구분코드 |itsmall| (상품구분을 위한 코드로 '|itsmall|' 로 입력 하시면 됩니다. 필수 사항입니다.)
  * 분류코드 100100100100/ (분류관리에서 등록된 분류코드+'/')
  * 상품명 아이폰애플 iPhone 4 (16GB)
  * 소비자가 200000 (숫자만 입력)
  * 판매가 150000 (숫자만 입력)
  * 제조사 애플
  * 원산지 한국
  * 브랜드 iPhone (등록된 브랜드가 없을 경우 자동으로 새로운 브랜드가 생성 됩니다.)
  * 모델명 iPhone 4G16
  * 적립금 B (A:없음, B:쇼핑몰 환경정책, 1~99 : 별도책정 ex. 5%경우 : 5)
  * 재고 99 (A:무제한, B:품절, C:상품숨김, 0~ : 한정판매 ex. 100개일 경우 : 100, 등록만 되고 출력이 안되어야 될때 : B)
  * 큰 이미지 http://itsmall.kr/images/10001.jpg (상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, )
  * 중간 이미지 http://itsmall.kr/images/10001.jpg (상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, )
  * 리스트 이미지 http://itsmall.kr/images/10001.jpg (상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, )
  * 작은 이미지 http://itsmall.kr/images/10001.jpg (상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, )
  * 디스플레이 이미지 http://itsmall.kr/images/10001.jpg (상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, )
  * 이미지 자동생성 Y (Y : 큰 이미지를 이용 다른 이미지 자동생성, N : 자동생성안함)
  * 상품 다른 이미지 http://itsmall.kr/images/10001.jpg|http://itsmall.kr/images/10002.jpg
(상품 이미지가 존재하는 URL, gif/jpg 이미지만 가능, '|'구분으로 다수 입력가능 )
  * 상품상세설명 상품에 대한 상세설명을 입력 (html을 등록 하셔도 됩니다.)
  * 옵션 색상,흰색,0,99,aaaaaa|색상,검정,1000,99,sssssss (옵션명,옵션항목,추가금액,재고수량,비고|옵션명,옵션항목,추가금액,재고수량,비고 . . . )
