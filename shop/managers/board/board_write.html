생성된 게시판 리스트를 확인 하거나 설정을 변경 하실 수 있습니다.
 
  * 게시판 이름     (게시판 이름은 수정이 불가능합니다.)     (영문, 숫자,'_' 만 가능)
  * 스킨 선택
  * 타이틀
  * 가로크기     게시판 가로크기 (100이하이면 %로 설정)
  * 페이지당 목록수     한페이지당 출력될 목록의 수 (1~999)
  * 페이지 표시수     목록의 아래부분에 표시될 페이지의 갯수 (1~999)
  * 배경색상     게시판 배경에 나타날 색상 (투명)  
  * 제목 글 제한수     리스트에 나타날 제목 글 제한수 ( 0 : 제한 안함)
  * 이미지 제한크기     글읽기에 나타날 이미지 제한크기 ( 0 : 제한 안함)
  * 웹에디터사용   웹에디터 기능사용
  * 전체목록 출력   글내용볼때 아래에 전체 리스트 출력 기능사용
  * 카테고리   카테고리 기능사용
  * HTML   HTML허용 기능사용
  * 공지사항   공지사항 기능사용
  * 자료실   자료실 기능사용     * 업로드 파일수 :
  * 간단한 답글   간단한 답글 기능사용
  * 비밀글   비밀글 사용안함   비밀글 사용자선택   비밀글 기본적용   (관리자와 비번 아는 사람만 볼수 있음)
  * 답변메일   답변글을 메일로 받을수 있음
  * 동영상 링크   링크된 동영상파일(플래쉬포함) 보기 기능사용
  * 관련사이트 링크   관련사이트 링크 기능사용    * 링크 가능수 :
  * 새글 표시 시간   등록 후 새글로 표시되는 시간
  * 내용글 제한수   방명록 형태의 리스트에 나타날 내용 글 제한수 ( 0 : 제한 안함 )
  * 방명록 형태   리스트시 글내용을 가지고 옵니다. 글수가 많을면 속도가 느려질 수 있습니다.
  * 블로그 형태   블로그 형태(글보기 부터 출력)를 사용 합니다. ( 블로그형태 스킨일 경우만 적용 )
  * 게시판 관리회원   게시판을 관리할 회원 아이디 ( 글 수정,삭제 가능 )
  * 갤러리 옵션 #1   갤러리일 경우 한줄에 보여줄 사진 수
  * 갤러리 옵션 #2   갤러리일 경우 보여줄 사진 사이즈 (가로)
  * 갤러리 옵션 #3   갤러리 사진 사이즈 비율로 고정 (4:3)
  * Header File   게시판 상단에 불러올 파일
  * Header Contents
  * Footer File   게시판 하단에 불러올 파일
