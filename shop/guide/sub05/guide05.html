SMS관리
01. SMS설정
A. SMS 아이디 : SMS신청시 등록한 아이디를 입력하시면 됩니다.
B. SMS 비밀번호 : SMS신청시 등록한 비밀번호를 입력하시면 됩니다.
C. 회신전화번호 : 고객이 받는 메세지에 회신번호로 찍히는 전화번호를 입력하시면 됩니다.
D. 관리자 핸드폰 : 관리자에게도 메세지를 발송할 때 사용할 핸드폰번호를 입력하시면 됩니다.

! SMS를 사용하실려면 'SMS 신청하기' 버튼을 누르신 후 회원 가입을 하기시 바랍니다.
! 회원 가입을 하시면 300캐쉬가 무료로 적립되며 문자 10건 정도를 테스트 해 보실 수 있습니다.
! SMS를 발송 하실려면 캐쉬가 있어야 되며 원하시는 금액을 충전하셔서 사용하시면 됩니다.(최소 3,000원 부터 충전이 가능하며 3,000원일 경우 136건의 문자를 발송 할 수 있습니다.)
! SMS를 신청후 정보를 입력 하시면 아래에 SMS충전정보가 나타납니다.
! SMS는 캐쉬가 충전되어 있어야 발송이 가능합니다.
! SMS발송 업체는 Coolsms 이며 자세한 사항은 http://www.coolsms.co.kr 에서 확인 하시기 바랍니다
02. SMS자동발송 관리
A. 회원가입시 발송 : 회원가입시 발송될 문구와 발송유무를 선택 할 수 있습니다.
B. 비밀번호찾기시 발송 : 비밀번호찾기시 발송될 문구와 발송유무를 선택 할 수 있습니다.
C. 무통장 주문완료시 발송 : 무통장 주문완료시 발송될 문구와 발송유무를 선택 할 수 있습니다.
D. 무통장 입금요청 발송 : 무통장 입금요청 발송될 문구와 발송유무를 선택 할 수 있습니다.
E. 입금확인시 발송 : 입금확인시 발송될 문구와 발송유무를 선택 할 수 있습니다.
F. 상품배송시 발송 : 상품배송시 발송될 문구와 발송유무를 선택 할 수 있습니다.
G. 주문취소시 발송 : 주문취소시 발송될 문구와 발송유무를 선택 할 수 있습니다.
H. 상품품절시 발송 : 상품품절시 발송될 문구와 발송유무를 선택 할 수 있습니다.


! 치환코드와 치환되는 값
{shopName} : 쇼핑몰명
{name} : 회원명 or 주문자명
{goodsName} : 상품명 (상품품절시 발송에만 가능)
{number} : 주문번호 (주문관련 발송에만 가능)
{price} : 결제금액 (주문관련 발송에만 가능)
{account} : 은행명 계좌번호 입금자명 (무통장 입금요청 발송에만 가능)
{deli_comp} : 택배사 (상품배송시 발송에만 가능)
{deli_no} : 송장번호 (상품배송시 발송에만 가능)
03. SMS주소록
쇼핑몰 회원과는 별도로 SMS발송시 이용할 주소록을 관리 할 수 있습니다.
A. 그룹 : 그릅을 선택하거나 신규그를을 생성할 수 있습니다.
B. 이름 : 이름을 입력하시면 됩니다.
C. 핸드폰번호 : 핸드폰번호를 입력 하시면 됩니다.
D. 성별 : 성별을 선택 하시면 됩니다.
E. 비고 : 기타 관련 사항을 입력 하시면 됩니다.
04. SMS발송
A. 보내는 사람 : 발신자번호에 나타날 번호를 입력하시면 됩니다.
B. 발송 방법 : 개별, 회원등급별, 전체회원, 주소록그룹별, 주소록전체 중 선택 하시면 됩니다.
C. 받는 사람 : 개별발송일 경우 수신자번호를 직접 입력해서 추가 하거나 주소록 및 회원검색을 이용하여 추가할 수 있습니다.
D. 회원등급 : 회원등급별발송일 경우 원하시는 등급을 선택 하시면 됩니다.
E. 주소록그룹 : 주소록그룹별발송일 경우 원하시는 그룹을 선택 하시면 됩니다.
F. 발송 리스트 : 발송될 수신자수가 나타납니다.
G. 예약발송 : 예약발송을 하실경우 사용함에 체크 하시고 날짜를 선택 하시면 됩니다.
H. 발송내용 : 좌측 파란색 영역에 발송내용을 입력하시면 됩니다.
! 문자수가 80Byte 이상일 경우 장문(LMS)발송으로 되며 발송건수는 2건으로 집계 됩니다
05. 메일발송 상세내용보기
A. 발송시간 : 발송시간이 나타 납니다.
B. 수신번호 : 수신번호가 나타 납니다.
C. 전송결과 : 전송결과가 나타 납니다.
D. 메세지 : 발송한 내용이 나타 납니다.
E. 비고 : 에러 메세지나 기타 네세지가 나타 납니다.
F. 처리상태 : 처리상태가 나타 납니다.
G. 발송타입 : 80Byte 이내일 경우 단문으로 81Byte 이상일 경우 장문으로 나타 납니다.
