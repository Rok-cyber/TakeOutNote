메일관리
01. 회원 메일링
A. 발송방법 : 개별, 회원등급별, 전체회원 중 선택 하시면 됩니다. 회원등급별을 선택 하시면 회원등급선택이 나타나며 원하시는 등급을 선택 하시면 됩니다. 회원등급별이나 전체회원은 메일 수신 허용회원에게만 발송 됩니다. 메일주소의 타당성은 발송중 처리되므로 총보낼 메일건수와 실제발송건수와 다를수 있습니다.
B. 받는사람 : 메일을 받는 사람의 주소를 입력 하시면 됩니다. 개별적으로 메일을 보낼 경우에만 나타 납니다.
C. 제목 : 메일제목을 입력 하시면 됩니다. 광고성 메일일 경우 반드시 (광고)라고 제목 앞부분에 입력 하셔야 됩니다.
D. 내용 : 메일내용을 입력 하시면 됩니다. 가로 사이즈 700이하로 작성하시기 바랍니다.
E. 메일링시작 : '메일링시작'을 클릭 하시면 메일링이 시작 됩니다. 대량으로 전송할 경우 메일전송 완료 창이 뜰때 까지 브라우저를 종료 하시면 안됩니다.
 
! 메일은 쇼핑몰 기본 메일폼을 이용 합니다.
! 메일을 100통이상 전송할경우 시스템에따라 수초내지 수분의 시간이 지연됩니다.
! 대량메일링은 시스템의 부하가 크므로, 자주 발송할 경우 관리자의 경고또는 조치가 취해질 수도 있습니다.

02. 메일 발송내역
전체 메일링이나 개별메일 내역및 전송 리스트를 확인 하거나 검색할 수 있습니다.
메일제목을 클릭 하시면 메일전송 상세내용을 볼 수 있습니다.

03. 메일발송 상세내용보기
A. 전송일 : 전송일이 나타 납니다.
B. 받는이 : 받는이가 나타 납니다. 단체 메일링일 경우 Mailling으로 검새결과메일링일 경우 Search Mailling로 나타 납니다.
C. 제목 : 제목이 나타 납니다.
D. 전송 결과 : 전송 결과가 나타 납니다. 전체 발송건수와 성공건수, 실패건수를 알 수 있습니다..
E. 처리 상황 : 처리 상황을 나타 납니다. 전송중단일 경우 '중단회원메일보내기'로 계속해서 메일링을 하실 수 있습니다.
F. 보낸 시간 : 메일을 처음 보낸 시간과 완료 시간 및 걸린 시간을 알 수 있습니다..
G. 에러 메세지 : 메일전송중 발생한 에러를 보여 줍니다. 전송 실패시 에러 메시지를 확인 해 보시면 실패 원인을 알 수 있습니다.
