가입양식설정
01. 입력양식 설정
A. 주민번호 : 주민번호를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다. 주민번호를 사용할 경우 주민번호 뒷자리는 암호화 되어서 저장되기 때문에 관리자라도 알 수가 없습니다.
B. 전화번호 : 전화번호를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
C. 핸드폰번호 : 핸드폰번호를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
D. 주소 : 주소를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
E. 홈페이지 : 홈페이지를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
F. 메신저 : 메신저를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
G. 생년월일 : 생년월일을 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.
H. 성별 : 성별을 사용안함, 사용함에서 선택 하시면 됩니다.
I. 결혼유무 : 결혼유무를 사용안함, 사용함에서 선택 하시면 됩니다.
J. 최종학력 : 최종학력을 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다. 최종학력 사용함이거나 필수항목일 경우 최종학력입력 부분에 학력사항을 입력 하시면 회원가입시 선택 할 수 있습니다.
K. 취미 : 취미를 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다. 취미 사용함이거나 필수항목일 경우 취미입력 부분에 취미사항을 입력 하시면 회원가입시 선택 할 수 있습니다.
L. 직업명 : 직업명을 사용안함, 사용함, 필수항목에서 선택 하시면 됩니다.

P. 추가필드1~5 : 그외 필요한 항목이 있을경우 추가필드에 필드명과 사용유무를 선택 하시면 됩니다.
! 아이디, 이름, 비밀번호, 이메일 주소는 필수입력 사항입니다.
02. 아이디제한 설정 : 아이디제한 설정을 사용할 경우 사용금지 아이디 입력한 아이디는 회원가입시 사용할 수 없습니다.
