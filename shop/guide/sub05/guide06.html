적립금관리
01. 적립금관리 리스트
적립내역 리스트를 확인 하거나 검색할 수 있습니다.
적립내용을 클릭 하시면 수정 할 수 있습니다.
적립아이디를 클릭 하시면 회원의 기본정보, 주문정보 등 자세한 정보를 확인 할 수 있습니다.
02. 적립금 등록하기
A. 적립 아이디 : 적립 아이디를 '회원검색하기'를 클릭 하셔서 이름으로 검색 하신 후 선택 하시면 됩니다.
B. 적립내용 : 적립내용을 입력 하시면 됩니다.
C. 적립금액 : 적립금액을 입력 하시면 됩니다.
D. 적립상태 : 적립대기, 적립완료, 적립사용, 적립취소에서 선택 하시면 됩니다. 적립대기는 상품주문 후 수령확인 전 까지의 상태 입니다. 적립완료는 회원에게 해당 금액만큼 적립금이 플러스되며 적립사용은 회원의 적립금에서 해당 금액만큼 적립금이 마이너스됩니다. 적립취소는 적립완료나 적립사용상태에서 원래대로 되돌려 줍니다.
 
! 쇼핑몰에서 이벤트 당첨회원이나 특별회원에게 적립금을 주실때 사용 하시면 됩니다.
