출석체크관리
01. 출석체크 리스트
출석체크 리스트를 확인 하거나 검색할 수 있습니다.
출석체크명을 클릭하면 출석체크 내용을 수정할 수 있습니다.


02. 출석체크 등록
A. 출석체크명 : 출석체크명을 입력 하시면 됩니다.
B. 진행기간 : 출석체크 진행기간을 입력 하시면 됩니다. 기간은 시작일 00:00분 부터 종료일 23:59분 까지 적용 받습니다. 출석체크 기간은 중복되게 설정 할 수 없습니다
C. 출석체크형태 : 연속 출석형, 횟수 출석형, 매일 출석형 중 선택 하시면 됩니다.
D. 적립금 지급방식 : 수동지급, 자동지급 중 선택 하시면 됩니다.
E. 출석혜택 : 출석체크 조건 만족시 지금될 포인트를 입력 하시면 됩니다.
F. 출석체크 방법 : 출첵 스템프(도장) 찍기, 출첵 댓글 달기, 회원로그인시 자동 중 선택 하시면 됩니다.
G. 메세지 : 각 항목별로 출력될 메세지를 입력 하시면 됩니다.
H. 타이틀 이미지 : 타이틀 이미지 이미지를 등록 하시면 됩니다. 위치는 카테고리 타이틀 이미지와 같고 스킨에 따라 사이즈가 다를 수 있으니 확인 해서 등록 하시기 바랍니다.
I. 삽입코드 : 페이지 상단에 나타날 디자인을 꾸미시면 됩니다.
03. 출석체크 참여 리스트
출석체크에 참여한 회원 리스트를 확인 하거나 검색 할 수 있습니다.
검색 조건을 이용하여 만족하는 참여 회원을 확인 할 수 있습니다.
