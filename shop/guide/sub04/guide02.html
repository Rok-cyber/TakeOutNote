메인디자인관리
01. 메인이미지관리 : 메인이미지를 등록 하시면 됩니다. 메인 이미지를 사용할 경우 사용함에 체크 하시고 이미지와 링크주소를 입력 하시면 됩니다.
02. 메인상품관리
A. 인기상품 : 메인페이지 인기상품 진열을 설정 하시면 됩니다. 사용하실경우 리스트형, 상품이동형, 상품그룹이동형 중 원하시는 형태를 선택 하시고 진열 갯수를 선택 하시면 됩니다.
B. 추천상품 : 메인페이지 추천상품 진열을 설정 하시면 됩니다. 사용하실경우 리스트형, 상품이동형, 상품그룹이동형 중 원하시는 형태를 선택 하시고 진열 갯수를 선택 하시면 됩니다.
C. 신상품 : 메인페이지 신상품 진열을 설정 하시면 됩니다. 사용하실경우 리스트형, 상품이동형, 상품그룹이동형 중 원하시는 형태를 선택 하시고 진열 갯수를 선택 하시면 됩니다.
D. 하단MBOX : 메인페이지 하단에 나타나는 MBOX((주간클릭베스트, 주간판매베스트, 베스트이용후기, 공지사항)의 사용유무를 선택 하시면 됩니다.
E. 브랜드 SHOP : 메인페이지 좌측 카테고리 밑에 나타나는 브랜드샵 박스 사용유무를 선택 하시면 됩니다. 사용 할 경우 등록된 브랜드 리스트가 나타 납니다.
