메일발송디자인관리
01. 쇼핑몰메일이미지 : 회원가입, 상품구매등 메일에 사용되는 이미지입니다. 사이즈를 맞게 작업하셔서 이미지나 플래쉬파일을 등록 하시면 됩니다. (플래쉬 파일일 경우 메일업체에 따라 나타나지 않는 경우가 있으니 참고 하시기 바랍니다.)
02. 회원가입축하문구 : 회원가입시 축하문구를 입력 하시면 됩니다. 자동발송여부에 자동으로 보냄으로 선택 하셔야 메일이 발송 됩니다.
03. 상품구매문구 : 상품 구매시 문구를 입력 하시면 됩니다. 자동발송여부에 자동으로 보냄으로 선택 하셔야 메일이 발송 됩니다.
04. 상품발송문구 : 상품 발송시 문구를 입력 하시면 됩니다. 자동발송여부에 자동으로 보냄으로 선택 하셔야 메일이 발송 됩니다.
