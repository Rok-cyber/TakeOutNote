기본문서관리
01. 회사소개 : 회사소개를 입력 하시면 됩니다. {shopName}은 자동으로 쇼핑몰명으로 치환 됩니다.
02. 이용약관 : 이용약관을 입력 하시면 됩니다. 기본등록된 이용약관을수정, 변경 하시면 됩니다. {shopName}은 자동으로 쇼핑몰명으로 치환 됩니다.
03. 개인정보취급방침 : 개인정보취급방침을 입력 하시면 됩니다. 기본등록된 개인정보치급방침을 수정, 추가 하시면 됩니다.
{shopName}은 쇼핑몰명, {name} 은 관리자명, {email}은 관리자 이메일, {tel}은 연락처로 자동 치환 됩니다.
개인정보취급방침 포함 내용은 아래 와 같습니다.
    ① 개인정보의 수집 및 이용목적과 수집항목 및 수집방법
    ② 개인정보 보유 및 이용기간
    ③ 제3자에게 제공하는경우 제공받는 자의 성명 또는 법인명칭 (해당하는 경우)
    ④ 개인정보를 위탁하는 경우 위탁업무의 내용 및 수탁자 (해당하는 경우)
    ⑤ 이용자 및 법정 대리인의 권리와 행사방법
    ⑥ 개인정보를 자동으로 수집하는 장치 (쿠키등)의 설치 및 거부에 대한 사항
    ⑦ 개인정보 관리 책임자의 성명, 연락처, 부서 (허위기재시 처벌)

04. 이용안내 : 이용안내를 입력 하시면 됩니다. 기본등록된 이용안내를 수정, 추가 하시면 됩니다.

! {shopName} : 쇼핑몰명, {email} : 관리자 이메일, {tel} : 연락처, {bankInfo} : 무동장 입금정보(예금주 은행명 계좌번호), {carrLimit} : 배송료적용 주문금액, {carrPrice} : 배송료로 자동 치환 됩니다.
! 정보통신망법 제27조의2 및 동법 시행령 제14조의3에 의거 개인정보취급방침의 전자적 표시를 적용 해야 됩니다.
개인정보취급방침 전자적 표시 적용하기
① 개인정보취급방침의 전자적 표시란?
이용자들이 웹 사이트의 개인정보취급방침을 간편하게 확인할 수 있도록 방침의 내용 및 형식을 규격화한 것입니다. 이용자는 본 "전자적 표시"를 자동 인식하는 프로그램을 통해 방침 내용을 일일이 읽지 않고도 핵심적인 사항들을 이미지 등으로 통해 손쉽게 확인할 수 있습니다.
② 전자적 표시 적용방법
한국정보보호진흥원 홈페이지(www.1336.or.kr)의 개인정보취급방침 작성하기 소프트웨어를 이용하여 전자적 표시를 손쉽게 작성하실 수 있습니다.
본 프로그램을 통해 자동 생성되는 ‘전자적 표시 파일‘을 다운로드 받아 귀사가 운영하는 홈페이지 웹 서버 루트 디렉터리에 올리시면 됩니다.
