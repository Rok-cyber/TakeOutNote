상/하단디자인관리
01. 로고관리 : 로고를 등록 하시면 됩니다. 등록시에 사이즈 맞게 수정 하신 후 올리시기 바랍니다.
02. 메뉴관리 : 상단메뉴 부분을 등록 하시면 됩니다. 플래쉬로 제작 되었을 경우 플래쉬를 선택 하시고 이미지로 제작 되었을 경우 이미지를 선택 하시면 됩니다. 이미지일 경우 최대 10개까지 등록 하실 수 있으며 전체 가로 사이즈만 지켜 주시면 됩니다. '메뉴수정'을 클릭 하시고 이미지와 링크주소를 입력 하시면 됩니다.
예) 전체 가로 사이즈가 700일 경우 5개를 등록 하려면 이미지 가로 사이즈를 140으로 하시면 됩니다.
03. 검색바배경이미지관리 : 검색바 배경이 있는경우 배경을 등록 하시면 됩니다. 스킨에 따라 없는 경우도 있습니다.
04. 하단정보관리 : 하단에 나타날 카피라이터 정보를 등록 하시면 됩니다. 이미지로 등록 하시려면 이미지 사용에 체크 하시고 이미지를 등록 하시면 됩니다. 이미지 사용안함을 체크하시면 쇼핑몰 기본정보를 이용해 텍스트로 나타납니다.
