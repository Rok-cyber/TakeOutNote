추가페이지관리
01. 추가페이지 리스트
추가페이지 리스트를 확인 하거나 검색할 수 있습니다.
페이지 제목을 클릭 하시면 추가페이지 내용을 수정 할 수 있습니다.
페이지 주소를 클릭 하시면 새창으로 쇼핑몰 추가 페이지가 뜹니다.
02. 추가페이지 등록
A. 페이지 제목 : 페이지 제목을 입력 하시면 됩니다.
B. Left Menu : 체크 하시면 고객센터의 좌측메뉴가 나타 납니다. 사용시 내용 입력부분 가로사이즈를 메뉴사이즈 만큼 줄이시기 바랍니다.
C. 게시판 연결 : 게시판을 연결할 경우 연결할 게시판을 선택 하시면 됩니다. 게시판은 게시판 관리에서 추가 하시면 됩니다.
D. 내용 : 추가 페이지에 들어갈 내용을 꾸미시면 됩니다. 게시판 연결일 경우 게시판 위에 나타날 타이틀이나 내용을 꾸미시면 됩니다.
