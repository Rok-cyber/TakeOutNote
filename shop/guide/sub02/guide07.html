브랜드관리
01. 브랜드 리스트
브랜드 리스트를 확인 하거나 검색할 수 있습니다.
브랜드명을 클릭하면 브랜드 내용을 수정할 수 있습니다.
02. 브랜드 등록
A. 브랜드명 : 브랜드명을 입력 하시면 됩니다.
B. 검색 TAG : 사용자 검색시 사용되는 TAG를 입력 하시면 됩니다.
C. 로고 : 로고 이미지를 등록 하시면 됩니다. 위치는 카테고리 타이틀 이미지와 같고 스킨에 따라 사이즈가 다를 수 있으니 확인 해서 등록 하시기 바랍니다.
D. 삽입코드 : 페이지 상단에 나타날 디자인을 꾸미시면 됩니다.
03. 브랜드상품리스트
각 브랜드별로 상품리스트를 확인 하실 수 있습니다..
'선택브랜드상품해제'를 누르시면 해당 브랜드 상품에서 제외되며 상품은 삭제 되지 않습니다.
'브랜드상품 등록'을 누르시고 해당 브랜드에 등록될 상품을 선택 하시면 됩니다.
