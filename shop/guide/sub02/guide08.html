기획전관리
01. 기획전 리스트
기획전 리스트를 확인 하거나 검색할 수 있습니다.
기획전명을 클릭하면 기획전 내용을 수정할 수 있습니다.
02. 기획전 등록
A. 기획전명 : 기획전명을 입력 하시면 됩니다.
B. 타이틀 이미지 : 타이틀 이미지 이미지를 등록 하시면 됩니다. 위치는 카테고리 타이틀 이미지와 같고 스킨에 따라 사이즈가 다를 수 있으니 확인 해서 등록 하시기 바랍니다.
C. 삽입코드 : 페이지 상단에 나타날 디자인을 꾸미시면 됩니다.
03. 기획전상품리스트
각 기획전별로 상품리스트를 확인 하실 수 있습니다..
'선택기획전상품해제'를 누르시면 해당 기획전 상품에서 제외되며 상품은 삭제 되지 않습니다.
'기획전상품 등록'을 누르시고 해당 기획전에 등록될 상품을 선택 하시면 됩니다.
