상품등록
01. 상품 분류 선택
A. 등록될 분류선택 : 등록될 분류를 선택 하시면 됩니다. 등록될 분류는 하위 분류가 없는 마지막 분류를 선택 하셔야 됩니다.  ('→'마크는 하위분류가 있음을 나타 냅니다. '→'마크가 없는 분류를 선택 하시면 됩니다.)
B. 브랜드샵 등록 : 브랜드 상품일 경우 해당 브랜드샵을 선택 하시면 됩니다.
C. 스페셜샵 등록 : 스페셜 상품일 경우 해당 스페셜샵을 선택 하시면 됩니다. 스페셜샵일 경우 복수 선택이 가능 합니다.
! 브랜드샵과 스페셜샵은 상품 환경설정에서 관리 하실 수 있습니다.
02. 상품 기본정보 입력
A. 상품번호 : 상품번호는 자동부여됩니다.
B. 상품명 : 상품명을 입력 하시면 됩니다. (필수 입력 사항입니다.)
C. 모델명 : 상품 모델명을 입력 하시면 됩니다.
D. 제조사 : 상품 제조사를 입력 하시면 됩니다. 자주 사용하는 제조사는 상품환경설정 제조사관리에서 관리(등록, 수정, 삭제) 하시면 됩니다.
E. 원산지 : 상품 원산지를 입력 하시면 됩니다. 자주 사용하는 원산지는 상품환경설정 원산지관리에서 관리(등록, 수정, 삭제) 하시면 됩니다.
F. 판매가격 : 상품 판매가격을 입력 하시면 됩니다. (필수 입력 사항입니다.)
G. 소비자 가격 : 상품 소비자가격(정상가격)를 입력 하시면 됩니다. 소비자가는 판매가격 높아야 되며 입력시 쇼핑몰 상품 상세페이지에 나타 납니다.
H. 판매가 대체문구 : 판매가격이 미정일떄 나타날 문구를 입력 하시면 됩니다. 예)전화문의, 상담후 협의...
I. 기본판매 단위 : 기본판매 단위를 선택 하시면 됩니다. 기본판매 단위는 상품환경설정 판매 단위관리에서 관리(등록, 수정, 삭제) 하시면 됩니다.
J. 기본판매 수량 : 기본(최소) 판매되는 수량을 입력 하시면 됩니다. 고객이 상품 구매시 기본판매 수량 이하로는 주문 할 수 없습니다.
K. 재고량 : 재고량을 '무제한', '품절', '상품숨김', '한정판매'에서 선택 하시면 됩니다. 품절일 경우 쇼핑몰 상품 상세페이지에 품절로 표시되며 고객이 구매 할 수 없습니다. 상품숨김일 경우 쇼핑몰에 나타나지 않습니다. 한정판매일 경우 수량을 입력 하시면 고객이 상품 구매시 구매 수량만큼 감소 하며 수량이 0일 경우 품절로 표시되며 고객이 구매 할 수 없습니다.
L. 적립금 적용 : 적립금을 '없음', '쇼핑몰환경정책', '별도 책정'에서 선택 하시면 됩니다. 쇼핑몰환경정책은 쇼핑몰환경설정의 적립금 부분을 적용 받습니다. 별도 정책일 경우 적립률(%)을 입력 하시면 됩니다.
M. 배송비 적용 : 비송비를 '없음', '쇼핑몰환경정책', '별도 책정'에서 선택 하시면 됩니다. 쇼핑몰환경정책은 쇼핑몰환경설정의 비송비 부분을 적용 받습니다. 별도 정책일 경우 상품구매시 추가될 배송비를 입력 하시면 됩니다. (별도 책정일 경우 회원등급별 설정에서 배송비 무료회원 이라도 배송비가 적용 됩니다.)
N. 상품 태그 : 상품과 관련된 태그를 입력 하시면 됩니다.상품태그는 ","로 구분되며 20개 까지 등록이 가능 합니다. (쇼핑몰 정책에 따라 사용자가 쇼핑몰 상품 상세페이지에서 등록 할 수 있습니다.)
O. 관련 상품 : 상품과 관련된 상품을 '상품검색'을 통해 추록 할 수 있습니다. 관련상품의 수는 제한이 없습니다. 추가된 상품은 'x'를 누르시면 삭제 됩니다.
03. 상품 옵션 입력
상품의 옵션이 있을경우 옵션명을 입력 하시고 옵션항목을 입력 하시면 됩니다. 옵션 등록 수 제한이 없습니다.
옵션명(예. 직접입력)을 선택 하시고 '옵션추가하기'를 누르시면 옵션상세 입력박스가 나타 납니다.
옵션 상세 입력 박스에 옵션명(예. 색상), 옵션항목(예. 노랑), 추가금액(예. 1000원), 재고수량(예. 100개), 노출(예. Y)을 입력 하시면 됩니다.
재고량은 상품재고량이 한정판매일 경우에만 연동되며 무제한일 경우 옵션재고량은 연동 되지 않습니다.(상품 무제한이라도 옵션재고 품절일 경우 품절로 나타 납니다.)
옵션명은 상품환경설정 > 옵션관리에서 자주 사용하는 옵션명을 등록 할 수 있습니다.
같은 옵션명을 기준으로 그룹으로 형성되며 순서가 적용 되어 사용자 페이지에 나타납니다.
04. 상품 전시 선택
A. 메인페이지 : 메인페이지 인기상품, 추천상품, 신상품에 진열될 상품을 선택 하시면 됩니다.
B. 분류메인페이지 : 분류메인페이지 인기상품, 추천상품, 신상품에 진열될 상품을 선택 하시면 됩니다.
C. 아이콘 선택 : 상품의 아이콘을 선택 하시면 됩니다. 아이콘은 상품환경설정 아이콘관리에서 관리(등록, 수정, 삭제) 하시면 됩니다.
! 상품전시는 메인디자인, 분류디자인 관리에서 해당 상품 사용시에만 나타 납니다.
05. 상품 이미지 업로드
A. 큰이미지 : 큰이미지를 사이즈에 맞게 작업 하셔서 올리시면 됩니다. 상품 상세페이지 확대보기를 클릭 했을경우 사용되는 이미지 입니다. (필수 입력 사항입니다.)
B. 중간이미지 : 중간이미지를 사이즈에 맞게 작업 하셔서 올리시면 됩니다. 상품 상세페이지에 사용되는 이미지 입니다.
C. 리스트이미지 : 리스트이미지를 사이즈에 맞게 작업 하셔서 올리시면 됩니다. 상품 리스트 이미지형태일때 사용되는 이미지 입니다.
D. 작은이미지 : 작은이미지 사이즈에 맞게 작업 하셔서 올리시면 됩니다. 상품 리스트 리스트형태일때 사용되는 이미지 입니다.
E. 디스플레이이미지 : 디스플레이 이미지 사이즈에 맞게 작업 하셔서 올리시면 됩니다. 메인 및 분류메인페이지 상품에 나타나는 이미지 입니다.
F. 자동 리사이즈 사용 : 체크를 하시면 큰이미지만 등록 하면 등록하지 않은 다른 이미지들은 큰이미지를 자동으로 사이즈 조정하여 등록 됩니다.
! 움직이는 Gif (Ani Gif) 이미지는 자동리사이징이 안됩니다. 리사이징을 시키면 스톱된 이미지로 리사이징됩니다.
06. 상품 다른이미지 업로드
상품의 다른 이미지가 존재하는 경우 이미지를 등록 시키시면 됩니다.
'파일찾기'를 클릭 후 올릴 이미지를 선택 하시고 '열기'를 클릭 하시면 이미지 리스트에 이미지가 등록 됩니다.
올릴 이미지를 선택 하실때 여러개의 이미지를 선택 하시고 '열기'를 클릭 하시면 동시에 여러개의 이미지가 등록 됩니다.
07. 상품 상세설명 입력
상품에 관한 상세설명을 입력 하시면 됩니다.
그림과 함께 곁들여서 설명하시면 더욱 효과적입니다.
