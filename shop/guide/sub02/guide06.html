공동구매관리
01. 공동구매 설정
A. 사용상태 설정 : 공동구매를 사용하시면 사용으로 사용하지 않으시면 미사용을 선택 하시면 됩니다,
B. 페이지 링크주소 : 공동구매 페이지 링크 주소 입니다.
C. 접급권한 설정 : 공동구매 사용시 접근 가능한 회원의 등급을 선택 하시면 됩니다.
D. 공동구매 종료일 : 공동구매 마감 후 구매가능 한 날짜 입니다. 상품등록시 Default 값으로 등록
E. 타이틀 이미지 : 타이틀 이미지를 등록 하시면 됩니다. 위치는 카테고리 타이틀 이미지와 같고 스킨에 따라 사이즈가 다를 수 있으니 확인 해서 등록 하시기 바랍니다.
F. 삽입코드 : 페이지 상단에 나타날 디자인을 꾸미시면 됩니다.
02. 공동구매 리스트
등록된 공동구매 상품 리스트를 확인 하거나 원하시는 상품을 검색을 통해 쉽게 찾을 수 있습니다
상품을 선택해서 일괄적으로 삭제 하실 수 있습니다.
상품 삭제시 이미 주문한 고객이 있으면 상담시 곤란한 경우가 있으니 주의 하시기 바랍니다.
유사한 상품을 등록 하거나 다른 분류에 같은 상품을 등록 할떄 상품복사를 사용 하시면 편리 합니다.
03. 공동구매상품등록
A. 판매가 : 상품의 판매가를 입력 하시면 됩니다.
B. 구매제한 수량 : 1인당 구매 제한수량를 입력 하시면 됩니다. 0일경우 무제한
C. 공동구매 시작일 : 공동구매 시작일을 입력 하시면 됩니다.
D. 공동구매 마감일 : 공동구매 마감일을 입력 하시면 됩니다.
E. 공동구매 종료일 : 공동구매 마감이후 고객이 주문할 수 있는 날짜를 입력 하시면 됩니다. 이후는 판매완료로 처리되고 주문 하실 수 없습니다.
F. 주문시점 : 성립후주문(공동구매), 선주문(소셜커머스) 중 선택 하시면 됩니다. 선주문으로 하시면 소셜커머스 처럼 주문/결제 이후 공구가 미성립시 주문이 취소 됩니다.
G. 신청 수량뱔 판매금액 설정 : 신청 수량에 따른 공동구매가를 입력 하시면 됩니다. '항목추가하기' 버튼을 누르시면 입력 항목이 나타납니다.

! 기존 상품등록과 거의 유사하며 다른 부분들만 설명해 놓았습니다.
04. 공동구매신청리스트
해당 상품의 공동구매 신청리스트를 확인 할 수 있습니다.
