이벤트관리
01. 이벤트 리스트
이벤트 리스트를 확인 하거나 검색할 수 있습니다.
이벤트 제목을 클릭하면 이벤트 내용을 수정할 수 있습니다.
진행상태는 이벤트 준비중, 진행중, 종료가 있으며 진행중일 경우 '진행중'을 누르시면 새창으로 쇼핑몰 이벤트 페이지가 뜹니다.
이벤트 기간이 지나면 이벤트는 자동 종료되고 해당 이벤트 상품들은 해제 됩니다.
02. 이벤트 등록
A. 이벤트 제목 : 이벤트 제목을 입력 하시면 됩니다.
B. 이벤트 기간 : 이벤트 시작일 과 종료일을 입력 하시면 됩니다. 기간은 시작일 00:00분 부터 종료일 23:59분 까지 적용 받습니다.
C. 제공할인률 : 이벤트 상품 할인률(%)을 입력 하시면 됩니다. 회원 할인률과 중복시 할인률이 큰 경우만 적용됩니다.
D. 추가적립금 : 이벤트 상품 추가 적립률(%)을 입력 하시면 됩니다. 회원 추가적립금과 중복시 적립금이 큰 경우만 적용됩니다.
E. 이벤트 상품 : 이벤트 적용 받을 분류나 상품을 선택 하시면 됩니다. 전체상품일경우는 전체 상품이 적용 받습니다. 특정분류일 경우 분류를 선택 하시고 '추가 하기'로 계속해서 추가 할 수 있습니다. 특정 상품일 경우 '상품검색하기'로 상품을 검색 하셔서 추가 하시면 됩니다. 추가된 분류나 상품은 'x'를 누르시면 삭제 됩니다.

F. 타이틀 이미지 : 타이틀 이미지를 등록 하시면 됩니다. 위치는 카테고리 타이틀 이미지와 같고 스킨에 따라 사이즈가 다를 수 있으니 확인 해서 등록 하시기 바랍니다.
G. 상단 삽입코드 : 페이지 상단에 나타날 디자인을 꾸미시면 됩니다.

! 한상품당 하나의 이벤트만 적용 가능하며 마지막으로 등록/수정된 이벤트가 상품에 적용 됩니다.
03. 이벤트상품관리
진행중인 이벤트에 한해 각 이벤트별 상품 리스트를 확인 할 수 있습니다.
'해제'를 누르시면 해당 이벤트 상품에서 제외되며 상품은 삭제 되지 않습니다.
특정분류로 추가된 상품일 경우 상품 해제후 해당 이벤트 정보를 수정 하면 다시 이벤트 상품이로 설정 됩니다.
