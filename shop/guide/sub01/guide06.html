기타정책
01. 회원인증 정책 : 회원가입시 회원 서비스 인증방법(가입시 자동승인, 관리자 승인시 가입)을 선택 하시면 됩니다.
02. 태그 정책 : 상품상세정보 페이지에서 사용자 태그 등록 가능 여부를 선택 하시면 됩니다.
03. 비회원 정보이용동의 : 비회원이 주문 할 경우 정보이용동의 출력 여부를 선택 하시면 됩니다.
! 회원가입 란 등 개인정보를 수집하는 화면에 개인정보 수집·이용 등에 대한 고객의 동의절차를 마련 해야 됩니다.(정보통신망법 제22조)
04. 자동배송완료 : 상품발송 후 고객이 수령확인을 하지 않았을 경우 자동으로 배송완료 처리 될 일수를 적으시면 됩니다.
05. 현금결제 추가DC : 주문된 상품을 현금으로 결제시 배송비를 제외한 금액에서 할인될 퍼센트를 입력 하시면 됩니다.
택배회사관리
사용하시는 택배회사, 택배조회주소를 등록 하시면 배송조회시 자동으로 해당 배송조회 페이지로 넘어가 배송추적을 하실 수 있습니다.
자주 사용하는 택배회사는 기본적으로 등록 되어 있습니다. 사용하는 택배회사만 남기시고 다른 회사는 삭제하시면 주문관리에서 관리 하기가 편 합니다.
사용하시는 택배회사가 기본으로 등록된 택배회사에 없을 경우 택배조회주소는 택배회사 홈페이지를 통해 확인 하셔서 입력 하시거나 pMall로 연락 주시면 가르처 드리겠습니다.
! 네이버 택배조회 페이지에서 택배조회주소를 확인 하셔도 됩니다.
배송/반품/환불정보
상품상세페이지 하단에 나타나는 배송, 교환/반품, 환불 안내 정보를 입력 하시면 됩니다.
배송/반품/환불정보는 설치시 기본으로 입력 되어 있으며 원하시는 부분을 수정 하시거나 다시 입력 하시면 됩니다.
