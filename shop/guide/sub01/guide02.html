결제 정책
01. 결제 방법 : 쇼핑몰에서 사용을 원하는 결제방법(무통장입금,신용카드, 실시간 계좌이체,핸드폰)을 선택하시면 됩니다.
02. 전자결제대행사 : 전자결제 사용시 전자결제대행사(KCP)를 선택 하시면 됩니다. (자세한 사항은 아래 전자결제서비스를 참고 하시면 됩니다.)
03. 상점ID/상점KEY : 전자결제대행사에 신청을 하시고 발급받은 값을 넣으시면 됩니다.
04. 카드결제 최소금액 : 카드 결제시 결제가능 최소금액을 입력 하시면 됩니다.
05. 할부 기간 : 카드 결제시 이용가능한 할부기간을 선택 하시면 됩니다.
06. 무이자 할부 여부 : 카드 결제시 무이자 할부결제 여부를 선택 하시면 됩니다. (무이자결제는 반드시 PG사와 계약체결 후에 사용해야 합니다!)
07. 무이자 할부 기간 : 무이자 할부 사용시 '무이자기간코드생성'을 누루시고 코드를 생성 하시면 됩니다.
08. 은행계좌번호 : 무통장입금 사용시 입금가능한 은행정보(은행명, 계좌번호, 예금주)를 입력 하시면 됩니다. 은행정보는 수량에 제한없이 등록 하실 수 있습니다.
09. 에스크로 사용 여부 : 에스크로 사용 여부를 선택 하시면 됩니다. (에스크 서비스는 별도 계약을 하셔야 됩니다.)
! 전자결제서비스 안내
KCP등 PG업체를 선정하여 계약된 업체를 선택하여 제공받은 정보를 입력합니다.
일반적으로 PG사와 계약을 맺고나면 PG사로부터 메일로 관련 정보를 받게 됩니다.
메일로 받은 ID, Key File 등을 각각 입력 하시면 됩니다.
PG사와의 계약은 승인까지 약 7일 정도 소요됩니다.


* KCP 전자지불서비스 안내 (서비스 신청 및 구비서류 안내)
신용카드, 계좌이체, 가상계좌, 휴대폰결제, 현금영수증은 KCP와 계약이 선행되야 합니다.
- 결제수단 설정 : 쇼핑몰에서 사용을 원하는 결제수단을 설정하시면 됩니다. ( KCP와 계약을 신청한 이후에 KCP에서 메일로 고객님께 SITE CODE와 SITE KEY 를 보내드립니다. )
- KCP Code : 상점ID 값을 입력하시면 됩니다.
- KCP Key : 상점KEY 값을 입력하시면 됩니다.


! 에스크로서비스 안내
전자상거래상에서 판매자와 구매자가 거래합의 후 상품배송 및 결제과정 중에 거래사고를 예방하기 위하여 제3자 (에스크로 사업자)가 거래대금의 입출금을 공정하게 관리함으로써 매매를 보호하고, 판매자와 구매자 간의 거래 안전을 도모하는 서비스입니다. (2006.4.1 전자상거래 소비자보호법에 따른 의무 시행)
- 에스크로 의무시행 거래 (법제화 부분) : 10만원 이상의 현금 거래 (기존 무통장입금 거래 포함)
- 에스크로 면제거래 : 신용카드로 구매하는 거래, 배송이 필요하지 않은 재화 등을 구매하는 거래(컨텐츠 등), 10만원 미만의 현금 거래
