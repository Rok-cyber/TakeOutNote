적립금&배송 정책
01. 적립금 사용 유무 : 쇼핑몰에서 고객이 상품 구매시 일정금액을 적립시켜 주문시 사용할 수 있도록 사용유무를 선택 하시면 됩니다. (단골고객을 만들기 위한 마케팅기본전략입니다.)
02. 회원 가입시 : 회원가입시 적립될 금액을 입력 하시면 됩니다.
03. 상품 구매시 : 상품구매시 적립될 퍼센트(%)를 입력 하시면 됩니다. (기본 적립금 설정 부분이며 상품등록에서 상품별 별도 적립금을 설정 하거나 회원등급관리에서 회원등급별 추가 적립금을 설정 할 수 있습니다.)
04. 최소사용 가능금액 : 적립금이 있는 회원이 상품구매시 최소사용 가능한 적립금을 입력 하시면 됩니다.
05. 배송비 사용 유무 : 배송비 사용 유무를 선택 하시면 됩니다. (기본 배송비 설정 부분이며 상품등록에서 상품별 별도 배송비를 설정 하거나 회원등급관리에서 회원등급별 배송비 무료를 설정 하실 수 있습니다.)
06. 배송비 적용금액 : 배송비가 주문합계 금액이 일정 금액 미만일 경우 적용 금액을 입력 하시면 됩니다.
07. 배송비 : 배송비를 입력 하시면 됩니다.
08. 추가배송비 지역설정 : 추가배송비 지역설정으로 도서산간 등 지역별로 배송금액을 지정할 수 있습니다. 입력 형태는 '시/구/군,시/구/군' (지역검색후 선택하시면 됩니다)로 입력 하시면 됩니다.
추가배송비 지역은 회원등급별 배송비 무료에 적용받지 않고 별도로 배송비가 부과 됩니다.
예) 기본배송비가 3,000원 (3만원 미만시)이고, 제주지역을 3,000원으로 설정했을 경우, 제주 구매자가 3만원 이상 구매했다면 배송비가 3,000원이 되고 3만원 미만으로 구매 하였다면 배송비가 6,000원이 부과 됩니다.
