SSL(보안서버)정책
01. SSL 사용 여부 : SSL(보안서버)를 사용 여부를 선택 하시면 됩니다. (서버에서 도메인에 SSL 보안 셋팅을 해야만 '사용함'을 선택 할 수 있으며 세팅되어있지 않으면 '사용함'을 선택 할 수 없습니다.)
02. SSL 포트번호 : SSL 포트번호를 적으시면 됩니다. 미입력시 기본포트 443으로 접속됩니다.
03. SSL 적용 페이지
A. 관리자 로그인 페이지에 보안접속(SSL) 적용하여 전송 : 선택시 관리자 로그인 페이지에 적용
B. 회원 로그인 페이지에 SSL 적용하여 전송 : 선택시 회원 로그인 페이지에 적용
C. ID/비밀번호 찾기 페이지에 SSL적용하여 전송 : 선택시 ID/비밀번호 찾기 페이지에 적용
D. 회원가입 페이지에 SSL적용하여 전송 : 선택시 회원가입 페이지에 적용
E. 회원정보수정 페이지에 보안접속(SSL) 적용하여 전송 : 선택시 회원정보수정 페이지에 적용
F. 주문서 작성 페이지에 보안접속(SSL) 적용하여 전송 : 선택시 주문서 작성 페이지에 적용
! SSL(보안서버) 안내
! SSL(보안서버) 사용시 중요 데이터 암호화를 통해 속도는 떨어지지만 데이타를 안전하게 전송시킬 수 있습니다.
! SSL(보안서버)은 별도 유료서비스이며 호스팅 서버 관리자에게 쇼핑몰 도메인에 SSL 보안서버 셋팅을 요청 하시면 됩니다.
! SSL(보안서버) 이용시 사용자 PC에 별도의 보안프로그램 설치가 필요 없으며 쇼핑몰에 설치된 SSL인증서를 이용하여 데이타를 엄호화해 전송 합니다.
