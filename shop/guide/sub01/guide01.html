쇼핑몰 사업자정보 & 기본 정보
01. 쇼핑몰 주소 : 쇼핑몰의 주소를 입력 하시면 됩니다. 예) http://itsmall.kr
02. 쇼핑몰명 : 쇼핑몰명을 입력 하시면 됩니다.
03. 상호 : 쇼핑몰을 운영하시는 회사의 상호를 입력 하시면 됩니다.
04. 대표자명 : 쇼핑몰을 운영하시는 회사의 대표자명을 입력 하시면 됩니다.
05. 사업자등록번호 : 쇼핑몰을 운영하시는 회사의 사업자등록 번호를 입력 하시면 됩니다.
06. 통신판매 신고번호 : 쇼핑몰을 운영하시는 회사의 통신판매 신고번호를 입력 하시면 됩니다.
07. 사업장 주소 : 쇼핑몰을 운영하시는 회사의 사업장 주소를 입력 하시면 됩니다.
08. 연락처 : 쇼핑몰을 운영하시는 회사의 연락처를 입력 하시면 됩니다.
09. 팩스번호 : 쇼핑몰을 운영하시는 회사의 팩스번호를 입력 하시면 됩니다.
10. 관리자명 : 쇼핑몰 관리자명을 입력 하시면 됩니다.
11. 관리자 이메일 : 쇼핑몰 관리자 이메일을 입력 하시면 됩니다.
12. 상단 타이틀 : 상단 타이틀은 브라우저 상단틀에 나오는 타이틀입니다. 검색엔지 키워드는 검색사이트에서 유용한 키워드를 입력 하시면 됩니다.
13. 검색엔진 키워드 : 포탈/검색사이트에서 검색했을때 나오는 키워드입니다. 쇼핑몰을 잘 나타내는 구체적인 상품품목이나 단어를 입력 하시면 됩니다. 예) 명품화장품, LCD 모니터
14. 실시간 검색어 입력 : 쇼핑몰에 스크롤되며 나타나는 실시간 검색어 입니다. 실시간 검색어는 24시간 이내의 검색어로 집계가 되며 검색이 없을 시 나타날 검색어를 ","구분하시고 10개까지 입력 입력 하시면 됩니다.
! 쇼핑몰 사업자정보 & 기본 정보를 입력 하시면 하단 카피라이터 부분에 아래처럼 나타납니다.(하단정보관리에서 이미지 사용안함일 경우)
