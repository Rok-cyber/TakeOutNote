접속통계
01. 년별통계 : 년도별로 방문자수와 페이지뷰수를 확인 할 수 있습니다.
02. 월별통계 : 해당년도의 월별로 방문자수와 페이지뷰수를 확인 할 수 있습니다.
03. 일별통계 : 해당월의 일별로 방문자수와 페이지뷰수를 확인 할 수 있습니다.
04. 시간별통계 : 해당일의 시간별로 방문자수와 페이지뷰수를 확인 할 수 있습니다.
05. BROWSER별통계 : BROWSER별로 방문자수를 확인 할 수 있습니다.
06. OS별통계 : OS(운영체제)별로 방문자수를 확인 할 수 있습니다.
07. 사이트별통계 : 접속사이트별로 방문자수를 확인 할 수 있습니다.
08. 접속경로별통계 : 접속경로별로 방문자수를 확인 할 수 있습니다.
09. 검색어별통계 : 검색어별로 방문자수를 확인 할 수 있습니다. 검색엔진을 통해 접속했을 경우에만 적용 됩니다.
 
! 방문자수는 방문자 아이피를 체크하여 하루에 한번의 카운팅만 합니다. 페이지뷰수는 체크없이 무조건 카운팅 됩니다.
