팝업관리
01. 팝업 리스트
팝업 리스트를 확인 하거나 검색할 수 있습니다.
팝업제목을 클릭 하시면 수정 할 수 있습니다.
02. 팝업 등록하기
A. 팝업 제목 : 팝업제목을 입력 하시면 됩니다.
B. 팝업 상태 : 사용, 일시정지에서 선택 하시면 됩니다. 일시 정지는 종료일이 남았지만 일시적으로 팝업을 사용하지 않을 경우 선택 하시면 됩니다.
C. 팝업 형태 : 하루에 한번만, 항상에서 선택 하시면 됩니다. 하루에 한번만일 경우 팝업창 하단에 '오늘 하루 이창 닫기'가 나타나고 사용자가 클릭하면 하루동안 더이상 팝업이 뜨지 않습니다.
D. 종료 날짜 : 종료 날짜를 선택 하시면 됩니다. 종료 날짜를 입력하지 않으면 팝업이 계속 나타 납니다.
E. 팝업창 위치 : 팝업창의 위치를 입력 하시면 됩니다.
F. 팝업창 크기 : 팝업창 크기를 입력 하시면 됩니다. 팝업내용에 맞춰 크기를 정하시면 됩니다.
G. 내용 : 팝업내용을 입력 하시면 됩니다.
