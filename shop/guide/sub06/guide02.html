게시판생성
01. 게시판 기본설정
A. 게시판 이름 : 게시판 이름을 입력 하시면 됩니다. 게시판 이름은 영문이나 숫자만 사용 할 수 있으며 한글이나 특수문자( ' . + - _ " ; ...)는 사용 할 수 없습니다.
B. 스킨 선택 : 사용하실 스킨을 선택 하시면 됩니다. 필수 게시판은 스킨을 변경 하실 수 없습니다.
C. 타이틀 :게시판 타이틀을 입력 하시면 됩니다.
D. 가로크기 : 게시판 가로 사이즈를 입력 하시면 됩니다. 100이상일 경우 px, 100이하면 %로 설정 됩니다.
E. 페이지당 목록수 : 한페이지당 출력될 목록의 갯수를 입력 하시면 됩니다.
F. 페이지 표시수 : 목록의 아래부분에 표시될 페이지의 갯수를 입력 하시면 됩니다.
G. 배경색상 : 게시판 배경에 나타날 색상을 입력 하거나 선택 하시면 됩니다. '투명'을 클릭 하시면 배경이 투명하게 나옵니다. '색상선택'을 이용 하셔서 선택 하시면 편리 합니다.
H. 제목 글 제한수 : 리스트에 나타날 제목의 제한 글수를 입력 하시면 됩니다. 0일경우 제한 하지 않습니다.
I. 이미지 제한크기 : 글읽기에 나타날 이미지 사이즈의 제한 사이즈를 입력 하시면 됩니다. 0일경우 제한 하지 않습니다. 레이아웃 보다 큰 이미지가 등록 되었을 경우 레이아웃이 깨져 나오므로 사이즈를 제한 하시면 됩니다.
02. 게시판 옵션설정
A. 웹에디터사용 : 웹에디터 스킨을 이용시에만 선택 하시면 됩니다.
B. 전체목록 출력 : 선택 하시면 글 내용볼때 아래에 전체 리스트가 나타 납니다.
C. 카테고리 : 카테고리가 있을경우 선택 하시면 됩니다.
D. HTML :선택 하시면 글등록 옵션에 HTML이 나타 납니다. HTML은 글 등록시 HTML을 사용 할 수 있게 해 줍니다.
E. 공지사항 : 선택 하시면 공지글을 작성 할 수 있습니다. 공지글은 글 등록 순서와 관계없이 글 목록 맨 처음에 나타 납니다.
F. 자료실 : 선택 하시면 글 등록시 파일을 등록 할 수 있습니다. 자료실 기능을 사용 하실 경우 업로드 파일수도 선택 하셔야 됩니다.
G. 간단한 답글 : 선택 하시면 글읽기에서 글에 대한 간단한 댓글을 달 수 있는 폼이 나타 납니다.
H. 비밀글 : 선택 하시면 글등록 옵션에 비밀글이 나타 납니다. 비밀글은 글 등록자와 관리자만 글을 읽을 수 있습니다.
I. 답변메일 : 선택 하시면 글등록 옵션에 답변메일이 나타 납니다. 답변메일은 사용자가 등록한 글은 읽고 누군가가 답변글을 등록 할 경우 원본글의 등록자의 이메일로 답변 내용이 전달 됩니다.
J. 동영상 링크 : 선택 하시면 글 등록시 동영상 링크가 나타 납니다. 동영상링크에 동영상 주소를 입력 하시면 글 읽기시 바로 동영상이 재생 됩니다.
K. 관련사이트 링크 : 선택 하시면 글 등록시 관련사이트 링크가 나나 납니다. 관련사이트 링크를 사용하실 경우 링크가능수도 선택 하셔야 됩니다.
L. 로그인 아이콘 : 선택 하시면 리스트 상단에 로그인 어이콘이 나타납니다.
M. 새글 표시 시간 : 글 등록 후 새글로 표시되는 시간을 입력 하시면 됩니다.
N. 내용글 제한수 : 방명록 형태의 리스트에 나타날 내용 글 제한수를 입력 하시면 됩니다. 0일경우 제한 하지 않습니다.
O. 방명록 형태 : 게시판 리스트에서 글 내용을 가지고 옵니다. 등록 된 글이 많을면 속도가 느려질 수 있습니다.
P. 게시판 관리회원 : 게시판을 관리할 회원 아이디를 입력 하시면 되니다. 해당 아이디의 회원은 게시판의 글을 수정, 삭제 할 수 있습니다.
Q. 갤러리 옵션 #1 : 갤러리 스킨일 경우 사용되며 한줄에 보여줄 사진 수를 입력 하시면 됩니다.
R. 갤러리 옵션 #2 : 갤러리 스킨일 경우 사용되며 리스트에 보여줄 썸네일 가로 크기를 입력 하시면 됩니다.
S. 갤러리 옵션 #3 : 갤러리 스킨일 경우 사용되며 리스트에 보여줄 썸네일의 비율로 4:3으로 고정 시킵니다.
03. 게시판 Header&Footer 설정
A. Header File : 게시판 상단에 불러올 파일명을 입력 하시면 됩니다.
B. Header Contents : 게시판 상단에 들어갈 내용을 입력 하시면 됩니다. HTML파일로 작성 하시면 됩니다.
C. Footer File : 게시판 하단에 불러올 파일명을 입력 하시면 됩니다.
D. Footer Contents : 게시판 하단에 들어갈 내용을 입력 하시면 됩니다. HTML파일로 작성 하시면 됩니다.
